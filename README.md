# NJ Transit Rail Service Reliability Dashboard

This project analyzes NJ Transit rail service using official GTFS data to understand
service frequency, peak period intensity, and weekday vs weekend differences.

## Key Insights
- Identifies routes with the highest weekday morning peak service
- Compares weekday and weekend service levels by route
- Categorizes trips by time of day to highlight peak demand periods

## Data Source
- NJ Transit Rail GTFS feed

## Tech Stack
- Python
- pandas
- matplotlib

## Outputs
- Weekday morning peak service by route
- Weekday vs weekend service comparison

This project mirrors internal transit performance and service planning dashboards.
